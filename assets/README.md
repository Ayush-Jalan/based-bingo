# Team Poster Image

Add your team poster image to this directory:

**File name:** `team-poster.jpg`

This single image should show all 8 BasedIndia team members.

## Image Requirements

- Format: JPG, PNG, or WebP
- Recommended size: 1000x1400px (portrait) or similar aspect ratio
- File size: < 1MB for optimal loading
- Should clearly show all 8 team members

## Member Numbering

When players submit their tweets, they'll be asked to enter a number (1-8)
to identify which team member they photographed. Make sure you have a clear
reference for which number corresponds to which team member in the poster.
